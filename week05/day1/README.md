# JavaScript Practice

This project is a simple JavaScript exercise to learn objects and arrays

## Tasks

- Create and manage product objects
- Understand object properties, methods, and nesting
- Demonstrate shallow vs deep copying
- Add and remove products
- Calculate total inventory value
- Use Object.keys, Object.values, and Object.entries
- Object.freeze and computed property names

## Functions

- `addProduct(product)` → adds a product to inventory
- `removeProduct(id)` → removes a product by id
- `listProducts()` → returns all products
- `calculateTotal(inventory)` → calculates total value (price × quantity)

## How to Run

Run the file with Node.js:

```bash
node main.js
```
